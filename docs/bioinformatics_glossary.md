# Bioinformatics Glossary

## CLL

Chronic Lymphocytic Leukemia. A blood cancer of mature B cells.

## BCR

B-cell receptor. A receptor on B cells. In CLL, malignant B cells are clonal and express the same BCR.

## SNP

Single nucleotide polymorphism. A DNA position where people can have different nucleotides.

## FASTQ

A sequencing file containing read sequences and quality scores.

## BAM

A binary alignment file containing reads mapped to a reference genome.

## Alignment

The process of placing sequencing reads onto the reference genome.

## Variant calling

The process of identifying differences between sample DNA and the reference genome.
